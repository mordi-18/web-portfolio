Project Overview

Interior parallax mapping is a rendering technique that creates the illusion of 3D interiors behind 2D window surfaces without requiring actual geometry, significantly reducing performance costs. This method displaces texture coordinates based on the view angle and a height map to simulate depth, allowing a single plane to display a room that changes perspective as the camera moves. The technique relies on cubemaps or atlas textures that are carefully captured from 3D scenes, ensuring no elements leak between faces to maintain the illusion.

The core of this project is a GLSL shader that projects cubemaps with parallax correction inside a quad, creating the illusion of a full 3D room at a fraction of the cost. The cubemaps are generated from fully rendered rooms and then applied to simple quads in real time.
