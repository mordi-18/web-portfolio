Project Overview

Logbait is a custom engine built in C++ and OpenGL, developed as part of an academic project with a classmate. It features a full deferred rendering pipeline with SSAO and shadow-mapped dynamic lighting, along with a multithreaded job system that enables asynchronous asset streaming for near-instant editor startup. The engine uses an Entity-Component-System (ECS) architecture to ensure efficient runtime performance and smooth editor interaction. It also includes a polished ImGui-based editor with tools such as a flycam, entity browser, real-time component editing, FPS graph, and logging system. Additionally, it supports a range of shader effects, including depth of field, outline blending, cubemap rendering, and stylized visual techniques.

Key Features

Entity-Component-System

Architecture to ensure efficient runtime performance and smooth editor interaction
